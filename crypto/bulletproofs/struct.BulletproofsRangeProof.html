    commitment: &PedersenCommitment,
    bits: usize,
    domain: &'static [u8]
) -> Result<(), Error>

Verifies that commitment is a Pedersen commitment of some value with an unsigned bit length bits. This is equivalent to proving that the value is an integer within the range [0, 2^bits) Function only works for bits = 8, 16, 32, 64.
